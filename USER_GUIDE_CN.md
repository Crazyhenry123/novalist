## 1. 快速开始

### 登录

在浏览器中打开 Novalist，您将看到登录页面，包含 Novalist 标志和两个输入框：

- **邮箱地址** —— 输入您注册时使用的邮箱
- **密码** —— 输入您的密码

点击 **登录** 按钮。验证通过后，您将进入主工作台。您的登录状态会在浏览器中保持，最长可维持30天无需重新登录。

如果看到错误提示，请检查邮箱和密码是否正确。密码要求至少8位，包含大写字母、小写字母和数字。

### 连接状态

登录后，工作台顶部会显示一个小指示灯：

- **绿色圆点** + "已连接" —— 表示您与服务器保持实时连接，可以开始创作
- **红色圆点** + "未连接" —— 表示连接中断。系统会自动尝试重连（最多5次）。如持续断连，请刷新页面

### 退出登录

点击页面右上角您的邮箱旁边的 **退出登录** 按钮。

## 3. 创建小说

小说创建表单收集 AI 智能体创作所需的全部信息。部分字段为必填，其余为可选（但提供越详细，创作效果越好）。

### 故事前提（必填）

这是最重要的字段。请写下您的故事核心创意——中心概念、吸引点或简短的故事大纲。前提越具体、越生动，AI 就越能理解您的创作愿景。

**好的前提包括：**
- 明确的主角或情境
- 核心冲突或悬念
- 让这个故事与众不同的独特元素

**示例：**

> 一个天赋平平的少年，在宗门大比中意外触发上古禁地的传承，获得了一部能吞噬万物修为的功法。从此他从最低阶的外门弟子开始，一路碾压天才、逆天改命。但他渐渐发现，这部功法的真正来历，牵扯到万年前仙界覆灭的惊天阴谋。

> 都市白领林沐在一次加班猝死后，重生回到了十年前的高考前夕。带着前世的记忆和一个神秘的系统面板，她决定不再做那个任人摆布的"好女孩"，而是要在商业、感情和人生中全面逆袭。

> 末世来临的第三年，丧尸进化出了智慧，开始建立自己的"文明"。前特种兵陈锋带着一支小队穿越废墟城市，却发现丧尸女王递来了一份和平协议。

**避免含糊的前提：**
- "一个关于修仙的故事"（太宽泛）
- "发生了很刺激的事"（没有具体内容）

### 类型

选择最适合您故事的类型。类型会影响 AI 智能体的世界观构建、情节设计、节奏把控和文笔风格。

**中国网络文学类型：**

| 类型 | 适合的故事 |
|------|-----------|
| **玄幻** | 修炼体系、异世界冒险、逆天改命、万族争霸 |
| **仙侠** | 修仙求道、门派争斗、飞升成仙、天地法则 |
| **武侠** | 江湖恩怨、侠义精神、武功秘籍、门派纷争 |
| **穿越** | 现代人穿越到古代或异世界，利用现代知识改变命运 |
| **重生** | 回到过去，凭借前世记忆避开陷阱、抓住机遇 |
| **都市** | 现代都市背景，异能、商战、逆袭 |
| **言情** | 感情线为主，甜宠、虐恋、双向暗恋、先婚后爱 |
| **耽美** | 男男感情故事，年上、年下、竹马、破镜重圆 |
| **系统/游戏** | 获得系统面板、穿越游戏世界、任务升级 |
| **宫斗/宅斗** | 后宫权谋、大宅门中的生存博弈 |
| **末世** | 灾变后的世界，丧尸、异兽、灵气复苏 |
| **军事** | 战争、军旅生活、热血报国 |

**经典国际文学类型：**

| 类型 | 适合的故事 |
|------|-----------|
| **西式奇幻** | 魔法体系、龙与地下城、史诗战争 |
| **科幻** | 未来科技、太空探索、人工智能 |
| **悬疑推理** | 案件调查、线索推理、真相揭露 |
| **惊悚** | 高风险、时间压力、悬念丛生 |
| **恐怖** | 恐惧、惊悚、心理或超自然恐怖 |
| **纯文学** | 人物深度、主题探索、精致文笔 |
| **历史** | 历史事件重构、古代风貌还原 |
| **青春校园** | 成长故事、校园恋爱、青春烦恼 |

### 叙事结构

选择故事的结构框架，这决定了情节在各章之间如何组织。如果不确定，**三幕式** 是最百搭的选择。

**经典结构：**

**三幕式**（默认）
最广泛使用的框架。故事分为三个部分：铺垫（介绍世界和角色）、冲突升级（不断加剧的阻碍和危机）、高潮与结局（最终对决和收尾）。适用于几乎所有类型。

**英雄之旅**
由约瑟夫·坎贝尔提出的12阶段神话冒险框架。主角接受冒险召唤、跨入陌生世界、经历重重试炼、遭遇生死磨难、最终蜕变归来。非常适合玄幻、冒险和成长类故事。

**救猫咪节拍表**
由布莱克·斯奈德创立的15节拍商业化结构。每个重要故事节点都有精确的位置安排——开场画面、催化剂、中点转折、"一切尽失"时刻和终局。能产出节奏紧凑、翻页欲强的故事。非常适合惊悚和商业小说。

**起承转合**
东亚传统四段式叙事结构。不强制要求核心冲突。故事通过"起"（导入）、"承"（发展）、"转"（意外转折）、"合"（和谐收束）自然流动。适合角色驱动型故事和意境类文学作品。

**弗赖塔格金字塔**
古典五幕戏剧结构：铺垫→渐进→高潮→回落→结局。强调单一戏剧高峰。适合纯文学和悲剧。

**中国网文结构：**

**升级流**
为网文量身打造的结构。境界划分清晰，每个阶段都有明确的修炼目标、对手和突破。从练气到筑基，从金丹到元婴，每次突破都给读者满足感。适合玄幻、仙侠、系统文。

**爽文节奏**
快速建立冲突，主角迅速反击打脸，高潮一个接一个。每3-5章一个小高潮，每个大段落一个大爆发。不拖沓、不啰嗦，让读者欲罢不能。适合都市逆袭、重生复仇。

**伏笔回收**
前期大量埋设伏笔和线索，中后期逐一揭晓和回收。每次回收都让读者产生"原来如此！"的顿悟感。环环相扣，逻辑严密。适合悬疑、推理和阴谋类故事。

**起伏流（虐后甜）**
大起大落的情感结构。先让主角经历痛苦和磨难，再给予更大的回报和甜蜜。虐得有深度、甜得有道理。适合言情、耽美和成长类故事。

### 写作风格

选择小说的文笔风格，这决定了 AI 写手的用词、句式和叙事手法。

**中国网文风格：**

| 风格 | 特点 | 适合 |
|------|------|------|
| **爽快打脸** | 节奏极快，冲突干脆利落，主角无敌但有趣。打脸情节密集，看了让人上头 | 玄幻、都市、重生、系统文 |
| **细腻情感** | 情感描写入微，擅长捕捉眼神、小动作和未说出口的话。感情线丝丝入扣 | 言情、耽美、慢热文 |
| **轻松搞笑** | 充满幽默和吐槽，角色互动妙趣横生。适当使用网络流行语，但不失深度 | 日常向、喜剧、欢乐冤家 |
| **热血燃文** | 激情澎湃、热血沸腾。面对绝境永不放弃，友情和信念的力量让人感动流泪 | 热血少年、军事、体育竞技 |
| **虐心催泪** | 误会、生离死别、求而不得。善于制造催泪情节，让读者边骂边追、边哭边看 | 虐恋、悲剧、BE向 |
| **古言雅韵** | 半文半白的典雅语言，适当穿插诗词。用词考究，意境悠远 | 古代言情、宫斗、仙侠 |
| **职场真实** | 充满行业专业术语和运作细节。让读者有"涨知识"的真实感 | 都市职场、医疗、法律 |

**经典文学风格：**

| 风格 | 特点 | 适合 |
|------|------|------|
| **商业流畅** | 干净利落的文笔，强钩子开头，节奏明快，通俗易懂 | 大众向类型小说 |
| **文学细腻** | 丰富隐喻、主题深度、句式复杂、心理描写精细 | 纯文学、获奖向作品 |
| **极简克制** | 短句为主，描写克制，大量留白，言外之意——鲁迅式冷峻 | 现实主义、黑色小说 |
| **华丽辞藻** | 辞藻华美，句式绵长，诗意盎然 | 玄幻、仙侠、历史 |
| **对话驱动** | 以对话推进故事，叙述极少 | 都市、喜剧、人物剧 |
| **快节奏动作** | 短段落、快切换、动感十足 | 惊悚、动作冒险 |
| **内心独白** | 深入内心、意识流手法、心理聚焦 | 心理悬疑、纯文学 |

### 叙事视角

选择读者通过谁的眼睛观察故事。

**第一人称**
叙述者是故事中的角色，使用"我"。读者只知道这个角色所知、所想、所感。适合制造悬念（不可靠叙述者）、成长故事和高度私人化的叙事。

**第三人称有限**（默认）
以第三人称叙述（"他"、"她"），但跟随一个角色的视角。读者能感知该角色的想法和情感，但不了解其他人的内心。最百搭的选择，适用于几乎所有类型。

**第三人称全知**
全知全能的叙述者，可以揭示任何角色的内心世界，也可以评论角色尚未知晓的事件。适合史诗玄幻、家族传奇和群像戏。

### 章节数

设定小说的章节数量，范围为3到50章。每章大约2000-4000字。参考：

| 章节数 | 大约总字数 | 阅读体感 |
|--------|-----------|---------|
| 3-5章 | 6千-2万字 | 中篇小说 |
| 8-15章 | 1.6万-6万字 | 标准长度小说 |
| 20-30章 | 4万-12万字 | 长篇小说 |
| 30-50章 | 6万-20万字 | 大长篇 |

默认的 **12章** 能产出一部节奏适中的标准长度小说。

### 角色设定（可选但推荐）

定义故事中的关键角色。每个角色可以填写：

- **姓名** —— 角色的名字（每个角色唯一必填项）
- **角色** —— 在故事中的功能：主角、反派、师父、盟友、红颜知己、对手、搞笑担当等
- **描述** —— 外貌特征、性格特点、背景或标志性特质
- **动机** —— 想要什么？被什么驱动？害怕什么？

点击 **+ 添加角色** 增加更多角色，点击角色旁的 **x** 按钮删除。

如果不填写角色，AI 会根据您的前提和类型自动创建合适的角色阵容。

**角色填写技巧：**
- 给主角一个明确的外在目标和一个内在矛盾
- 给反派一个"他觉得自己没错"的合理动机
- 即使简短的描述也有帮助："林默，主角，性格隐忍但内心骄傲的废材少年，要证明自己不比任何天才差"

### 世界设定（可选）

描述故事发生的世界背景。这是自由文本，可以包括：

- 修炼体系（练气→筑基→金丹→元婴→化神→渡劫…）
- 时代背景（远古、中世纪、现代、未来）
- 关键场所（宗门、秘境、皇宫、都市）
- 世界规则（灵气浓度、科技水平、社会结构）
- 势力分布（宗门等级、家族格局、国家对立）

留空则由 AI 世界构建师根据前提自动设计。

### 主题方向（可选）

建议您希望小说探索的深层主题。主题赋予故事超越表面情节的意义。中国网文常见主题：

- "逆天改命——命运是否可以被改写"
- "强者之路——站在巅峰之后还剩什么"
- "守护与牺牲的代价"
- "复仇之路上是否会迷失自我"
- "穿越重来一次，能否修正人生的遗憾"
- "末世中人性的真实面目"
- "爱情能否跨越身份和种族的鸿沟"

留空则由 AI 故事架构师根据前提挖掘最有力的主题角度。

### 基调

用几个词描述小说的整体情感氛围。默认是"引人入胜、沉浸感强"。更多示例：

- "暗黑压抑、步步惊心"
- "轻松搞笑、偶有深度"
- "热血沸腾、激情澎湃"
- "甜到齁、糖分超标"
- "虐心虐肺、刀子密集"
- "古朴典雅、意境深远"
- "紧张刺激、一刻不停"

### 开始生成

填写完至少"故事前提"后，点击紫色的 **开始生成** 按钮。按钮会变成"生成中..."，表单在 AI 智能体工作期间将被禁用。您可以在下方的智能体动态面板中实时观看创作过程。

## 4. 认识 AI 智能体团队

Novalist 使用六位专业 AI 智能体，以流水线方式协作完成小说创作。每位智能体在小说创作的某个环节拥有深厚专业能力。

### 第一阶段 — 基础构建（三个智能体并行工作）

**📐 故事架构师**（琥珀色）
设计小说的结构蓝图。根据您选择的叙事结构，创建：
- 小说的暂定标题
- 抓住故事核心意义的主题陈述
- 逐幕/阶段的故事走向分解
- 逐章计划，描述每章发生什么
- 定义叙事弧线的3-5个重大转折点

**👤 角色开发师**（绿色）
把您的角色简介（或从零创建角色）开发成完整的文学人物：
- 详细的背景故事
- 双重动机：外在想要什么 + 内在真正需要什么
- 角色成长弧线
- 语言特征和说话风格
- 角色关系网

**🌍 世界构建师**（蓝色）
构建故事所处的世界：
- 生动的世界概述（五感描写）
- 关键场景和地点
- 世界规则和力量体系
- 氛围基调
- 文化和社会细节

### 第二阶段 — 情节编排

**🧵 情节编织师**（紫色）
接收前三位智能体的全部成果，编织成详细的逐场景路线图：
- 情感和情节节拍对照表
- 每章的场景大纲（视角、目的、事件、情绪）
- 副线追踪
- 伏笔埋设和回收计划
- 整体张力曲线

### 第三阶段 — 正文创作

**✍️ 文笔写手**（粉色）
系统的创意核心。按照情节编织师的场景大纲，以您选择的风格生成章节正文：
- 生动的感官细节
- 有辨识度的角色对话
- 情感共鸣
- 场景内的节奏掌控
- 每章约2000-4000字

### 第四阶段 — 精修打磨

**📝 编辑**（青色）
审阅写手产出的每一章，打磨至出版品质：
- 消除陈词滥调、收紧文笔
- 调整节奏
- 检查风格和角色一致性
- 排查连贯性错误
- 强化开头钩子和结尾悬念

## 5. 实时观看创作过程

点击 **开始生成** 后，表单下方会出现"智能体动态"面板。这是一个实时自动滚动的信息流，显示每个智能体的工作状态。

### 您将看到

1. **流水线启动** —— 确认生成已开始，附带唯一的小说ID
2. **第一阶段启动** —— 故事架构师、角色开发师和世界构建师同时开工
3. **智能体成果** —— 每个智能体完成后显示：图标、彩色名称、完成状态和成果预览
4. **阶段递进** —— 从第一阶段（基础构建）依次推进到第四阶段（编辑润色）
5. **小说完成** —— 全部智能体完成工作的最终确认

### 理解信息流

每条消息包含：
- **图标**（📐 👤 🌍 🧵 ✍️ 📝）标识智能体身份
- **智能体名称**以专属颜色显示
- **类型标签**显示消息类别

- 可选的**预览框**（深色背景）展示智能体输出的样本

信息流自动滚动到最新消息。您可以随时向上滚动查看之前的动态。

### 生成时间

一部12章小说通常需要几分钟完成。进度动画和"AI 智能体正在协作创作您的小说..."提示会在全程显示。

如果某个环节出错，信息流中会显示红色错误消息。您可以修改输入后重试。

## 8. 创作秘籍：如何获得最佳效果

### 写一个好前提

前提是最重要的输入。以下是写出好前提的方法：

1. **要具体**。"一个修仙故事"太宽泛。"一个只有伪灵根的少年，在被所有宗门拒之门外后，意外获得了一颗来自上古大能的储物戒指，里面封印着一位只剩一缕残魂的绝世强者"——这才给 AI 足够的素材。

2. **包含冲突**。每个故事都需要张力。确保前提中有问题、有悬念或对立力量。

3. **点明独特之处**。什么是这个故事的创新点？什么让它与同类作品不同？

4. **2-4句话最佳**。足够具体，又留有 AI 发挥创意的空间。

### 网文爆款前提公式

**逆袭公式：** [弱势身份] + [意外获得/觉醒] + [在某个领域崛起] + [隐藏的更大秘密]

> 被当作废物嘲笑了十八年的林枫，在家族大比被暗算至重伤时，意外觉醒了传说中早已失传的混沌体质…

**重生公式：** [前世遗憾/惨死] + [重生时间点] + [这次要…] + [但发现…]

> 上一世被闺蜜和渣男联手害死的苏念，重生回到了大学入学第一天，这次她要先下手为强…

**穿越公式：** [现代身份] + [穿越到] + [利用现代知识/技能] + [意外卷入]

> 顶级外科医生穿越到了缺医少药的古代战场，一手神乎其神的手术技巧，让他被两国君主争相拉拢…

### 角色填写技巧

- **给主角取名**。有名字的角色比"主角"更鲜活
- **写动机比写外貌重要**。"想要找回被偷走的传承"比"身材修长、面容冷峻"更有用
- **创造角色之间的关系**。如果主角和反派有过往，请提到
- **留出惊喜空间**。不必完全定义每个角色，简短的提示就够 AI 发挥

### 基调的魔力

基调字段的影响力超乎想象。对比：

| 基调 | 对小说的影响 |
|------|------------|
| "暗黑压抑" | 沉重氛围、内心挣扎、道德灰色 |
| "甜到齁" | 高糖情节、互撩日常、温馨甜蜜 |
| "热血燃魂" | 宏大场面、激昂战斗、信念与友情 |
| "虐心虐肺" | 误会不断、生离死别、催泪名场面 |
| "诙谐吐槽" | 逗趣对话、无厘头日常、笑中带泪 |
| "古朴深远" | 古典意境、诗意表达、厚重历史感 |

## 9. 常见问题

**生成一部小说需要多长时间？**
一部12章小说通常需要几分钟。更长的小说（30章以上）耗时更长。您可以通过智能体动态面板实时跟踪进度。

**可以中途停止生成吗？**
目前点击"开始生成"后，流水线会运行到完成。如果遇到错误，会出现错误提示，您可以用不同的输入重试。

**为什么出现了错误？**
错误可能源于生成过程中的网络连接中断或 AI 处理问题。请刷新页面后重试。如果连接指示灯显示红色，请等待重连（变绿）后再提交。

**可以编辑生成的文字吗？**
当前版本以只读方式展示生成的小说。您可以选中并复制任何文字，粘贴到您常用的写作工具（如 WPS、Word、石墨文档等）进行进一步编辑。

**我的小说保存在哪里？**
您的小说保存在您的账户中。每部小说及其章节都安全存储并关联到您的用户ID。

**可以生成多部小说吗？**
可以。每次填写表单并点击"开始生成"都会创建一部新小说。之前的小说仍然保留。

**必须填写所有字段吗？**
不用。只有"故事前提"是必填的。其他字段都有合理的默认值。但您提供的信息越多，创作结果就越贴合您的愿景。

**如果不满意结果怎么办？**
尝试调整输入：
- 用更具体的细节完善前提
- 换一种叙事结构
- 更改写作风格
- 添加更多角色细节或世界设定
- 调整基调

即使是前提或基调的微小变化，也可能产生截然不同的结果。

**"类型"和"写作风格"有什么区别？**
类型决定写*什么样*的故事——影响情节惯例、世界观和读者预期。写作风格决定*怎样*写故事——文笔调性、句式结构和叙事技巧。同样是玄幻小说，"爽快打脸"风格和"华丽辞藻"风格读起来完全不同。

**"结构"和"章节数"有什么区别？**
结构定义故事的戏剧形状——转折点在哪里、张力如何起伏。章节数定义故事的长度。同样是"升级流"结构，8章是紧凑的短篇，30章是宏大的长篇。

**Novalist 使用什么AI模型？**
Novalist 使用 Anthropic 公司的 Claude Sonnet 4.6，通过 Amazon Bedrock 调用。不同智能体使用同一模型但不同的温度参数——规划类任务用低温度保证结构性，创作类任务用高温度激发创意。
